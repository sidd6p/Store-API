# Store-API
Flask API for Store Management



## Run Locally


Clone the project

```bash
  git clone https://github.com/sidd6p/Store-API.git
```

Go to the Store-API directory
```bash
  cd Store-API
```

Install dependencies

```bash
  pip install -r requirements.txt
```

run the project

```bash
  flask run
```



## Feature

- Item
